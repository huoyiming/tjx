# 企者不立，跨者不行······

{% hint style="info" %}
注意：此条目的解释可能与一般理解有所出入
{% endhint %}

### 原句

企者不立，跨者不行,自见者不明,自是者不彰,自伐者无功,自矜者不长。其在道也，曰余食赘形,物或恶之。故有道者不处。
